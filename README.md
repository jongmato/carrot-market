# 당근 마켓 클론 프로젝트

create-next-app@latest --typescript 를 사용하용하여 프로젝트를 생성하였습니다.

## ⚙️ Tools 

### Framework
- NextJs

### Language
- Typescript

### Lib
- ReactJs 18
- TailwindCSS
- Prisma
- SWR

### ETC.
- eslint
- prettier
- commitlint

## 구성
- enter
  - Login 화면
  - email, phone 선택 사능
  - SNS Login 기능
- items
  - 중고 거래할 물품(item) 게시물 리스트를 보여주는 화면
  - upload 하는 화면
- community
  - 유저가 올린 댓글(게시글) 리스트를 보여주는 화면
  - 댓글 상세 페이지 
  - 글 삭정 페이지
- chats
  - 유저들이 서로 대화할 수 있게 채팅 리스트를 보여주는 화면 (채팅)
